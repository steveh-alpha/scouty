# Scouty Landing Page (Testing Brand)

This is a market validation landing page for **Scouty**, a temporary testing brand used to validate the AI-concierge summer camp concept.

## 🎯 Purpose
*   **Brand Protection:** Uses "Scouty" instead of the final brand "Kivvi" to allow for messy testing and iteration without affecting brand equity.
*   **Audience:** Parents in the Philadelphia Suburbs (Chester, Delaware, and Montgomery counties).
*   **Validation Goal:** Measure the conversion rate of the $19 "AI Concierge" service vs. a "Free" directory.

## 🛠️ Design Changes
*   **Rebranded:** All references to "Kivvi" changed to "Scouty."
*   **Age-Agnostic:** Removed specific references to "8-year-old" or "ages 7-9" to prevent alienating parents of younger or older children. Used "age groups" and "your children" instead.
*   **Simplified UI:** Switched to a Blue-based professional color palette to differentiate from the final Kivvi brand identity.

## 🚀 How to Run
1.  Open `index.html` in any browser.
2.  Test the "Fake Door" strategy by clicking the pricing buttons.
3.  Observe the waitlist capture modal.

## 📊 Success Metrics
*   **Click-Through Rate (CTR):** Percentage of users who click a CTA.
*   **Lead Quality:** Number of emails captured in the "2026 Waitlist" modal.
*   **Service Preference:** Track which pricing tier (Free vs. $19) generates more initial clicks.
